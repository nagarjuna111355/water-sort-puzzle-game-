# Water Sort Puzzle Game

A fun and addictive HTML5 puzzle game where players sort colored water into matching bottles.

## 🎮 Game Description

Water Sort Puzzle is a brain-teasing puzzle game where your objective is to sort colored water into individual bottles. Each bottle can only contain one color of water, and you can only pour water on top of the same color or into empty bottles.

## 🚀 Features

- **Multiple Levels**: Progressive difficulty with increasingly complex puzzles
- **Move Counter**: Track your moves and try to complete levels with minimum moves
- **Timer**: Challenge yourself to complete levels as quickly as possible
- **Undo System**: Undo your last move if you make a mistake
- **Level Skip**: Skip difficult levels (limited skips available)
- **Infinity Mode**: Unlock unlimited skips for endless gameplay
- **Dynamic Bottle Management**: Add or remove bottles during gameplay
- **Profile System**: Track your progress, stars, and high scores
- **Settings**: Customize sound, vibration, theme, and difficulty
- **Responsive Design**: Works on desktop and mobile devices

## 🎯 How to Play

1. **Select a Bottle**: Click on any bottle to select it
2. **Pour Water**: Click on another bottle to pour the selected water
3. **Rules**:
   - You can only pour water on top of the same color
   - You can pour into empty bottles
   - Each bottle can hold up to 4 units of water
   - Complete the level when all bottles contain only one color

## 🎮 Game Controls

### In-Game Controls
- **Click/Touch**: Select and pour water between bottles
- **Undo Button**: Revert the last move
- **Reset Button**: Restart the current level
- **Home Button**: Return to the main menu
- **+/- Buttons**: Add or remove bottles (for custom challenges)

### Main Menu
- **Play Now**: Start a new game from level 1
- **Continue Level**: Resume from your last played level
- **Instructions**: View detailed game instructions
- **Profile**: View your statistics and progress
- **Settings**: Customize game preferences

## 🏆 Scoring System

- **Stars**: Earn 1-3 stars based on performance
- **Moves**: Fewer moves = higher score
- **Time**: Faster completion = higher score
- **High Scores**: Track your best performance for each level

## ⚙️ Settings

- **Sound**: Toggle sound effects on/off
- **Vibration**: Enable/disable haptic feedback (mobile)
- **Theme**: Choose visual theme
- **Difficulty**: Select game difficulty level
- **Auto Save**: Automatically save progress
